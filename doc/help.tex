\section{Modules and Libraries}
To extend Python's functionality using a library or module, simply do:
\texttt{import modulename}
\begin{itemize}
\item \texttt{math} - Mathematical functions, and constants like \texttt{pi}
\item \texttt{NumPy}
\item \texttt{scipy.ndimage}
\item \texttt{scipy.optimize}
\item \texttt{scipy.cluster}
